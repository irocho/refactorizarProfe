# Anotacións de refactorización
Para practicar en contornos no 3º trimestre o apartado de refactoring
### Chuletas rápidas
> Lembramos que:

-  _ctrl D_ atopa duplicados de palabras: **RENOMEAR** é a técnica de refactorización máis evidente.
-  _F1/ formato_ axuda a organizar visualmente o código.
- _Ctrl ç_ para que comente liñas ou bloques de código.
